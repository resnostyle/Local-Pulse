Local Pulse has been deployed.

1. Get the application URL:
{{- if .Values.ingress.enabled }}
  http://{{ .Values.ingress.host }}
{{- else }}
  export POD_NAME=$(kubectl get pods -l "app.kubernetes.io/component=go,app.kubernetes.io/instance={{ .Release.Name }}" -o jsonpath="{.items[0].metadata.name}")
  kubectl port-forward $POD_NAME 8080:8080
  Then visit http://localhost:8080
{{- end }}

2. Celery workers are processing scrape tasks. Check status:
  kubectl get pods -l "app.kubernetes.io/component=celery-worker,app.kubernetes.io/instance={{ .Release.Name }}"
  kubectl logs -l "app.kubernetes.io/component=celery-worker,app.kubernetes.io/instance={{ .Release.Name }}" --tail=50

3. Celery beat is scheduling tasks per source interval. Check status:
  kubectl get pods -l "app.kubernetes.io/component=celery-beat,app.kubernetes.io/instance={{ .Release.Name }}"

4. To use GHCR images, set in values:
   image.go.repository: ghcr.io/YOUR_ORG
   image.python.repository: ghcr.io/YOUR_ORG
